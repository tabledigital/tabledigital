# tabledigital

## Synopsis

Information and contact site for [table.digital](https://table.digital)

## Installation

Make sure you have [node](https://nodejs.org/) installed
```
node -v
```

Install gulp (if required)
```
npm i -g gulp-cli
```

Install dependencies
```
npm i
```

Run build
```
gulp
```
